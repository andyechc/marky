# Marky - Online Editor Markdown

Un editor de markdown online gratuito con interfaz moderna, vista previa en tiempo real y todas las herramientas que necesitas para escribir documentación de calidad.

## ✨ Características

- 🎨 **Diseño Moderno** - Interfaz limpia y profesional con TailwindCSS
- 🌓 **Tema Oscuro/Claro** - Cambia entre temas según tu preferencia
- ⚡ **Vista Previa en Tiempo Real** - Ve cómo se ve tu markdown mientras escribes
- ⌨️ **Atajos de Teclado** - Acelera tu escritura con atajos inteligentes
- 💾 **Auto-guardado** - Tu trabajo se guarda automáticamente
- 📱 **Responsive** - Funciona perfectamente en móviles y escritorio
- 📤 **Exportación** - Exporta a HTML y descarga tus archivos
- 📊 **Estadísticas** - Contador de palabras, caracteres y tiempo de lectura
- 🔖 **Archivos Recientes** - Acceso rápido a tus archivos recientes

## 🚀 Atajos de Teclado

| Atajo | Función |
|-------|---------|
| `Cmd+S` | Guardar archivo |
| `Cmd+B` | Texto en negrita |
| `Cmd+I` | Texto en cursiva |
| `Cmd+E` | Código |
| `Cmd+K` | Enlace |

## 🛠️ Tecnologías

- **React** - Interfaz de usuario moderna
- **Vite** - Build tool rápido y optimizado
- **TailwindCSS** - Framework de CSS moderno
- **Lucide React** - Iconos hermosos y consistentes
- **React Markdown** - Renderizado de markdown
- **Syntax Highlighter** - Coloreado de código

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/andyechc/marky.git

# Entrar al directorio
cd marky

# Instalar dependencias
bun install

# Iniciar desarrollo
bun run dev

# Construir para producción
bun run build
```

## 🌐 Uso

1. Abre tu navegador en `http://localhost:5173`
2. Comienza a escribir en el editor
3. Usa los atajos de teclado para acelerar tu trabajo
4. Cambia entre vista de editor y vista previa
5. Exporta tu trabajo cuando termines
